# Capital Services - Flask Project

Flask Project for Capital Services (BU MET CS 634);

## Getting Started
```
git clone https://github.com/ckarjadi/capital_services
cd capital_services/
flask run
```

### Prerequisites
```
+Python 3.6.5+

Click==7.0
Flask==1.1.1
itsdangerous==1.1.0

python-dotenv==0.11.0

```
